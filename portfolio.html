Below are links to some of the major projects I've worked on over the past five years.

Google Live Case

Smart, snappy site that allows users to design custom cases for their Nexus phones using either maps or photos from their desktop computer or mobile device.

  • Technologies: Angular 1.3, WebGL, ThreeJS, Browserify, Python-based AppEngine back end.
  • Notable features: Interactive 3D model, fully responsive design, based heavily on Material Design guidelines, several third-party integrations.
  • See it in action: Google Live Case

YouTube Creator Playbook

I was the lead developer on this project.

  • Technologies: Closure Javascript library, Bourbon, Neat, semantic markup optimized for SEO.
  • Notable features: Fully responsive design, based heavily on Material Design guidelines.
  • See it in action: YouTube Creator Playbook

Think With Google

  • Technologies: Closure Javascript library, Bourbon, Neat, semantic markup optimized for SEO.
  • Notable features: Localized for three markets, fully responsive design, based heavily on Material Design guidelines.
  • Accolades: Winner of the 2015 People's Voice Webby award for business products & services.
  • See it in action: thinkwithgoogle.com

Recommind User Interface Toolkit

  • Technologies: Backbone, Marionette, jQuery, RequireJS, Compass.
  • Notable features: User interface components used across the organization by different engineering departments, including modal dialogs, notifications, hierarchical lists, grid views, tooltips, etc. Components were written in so they were easily extensible.
  • See it in action: The UITK is used extensively in Axcelerate 5, the product featured in the video at left.

Recommind Decisiv Search

  • Technologies: XSLT, jQuery, JavascriptMVC, plain ol' CSS. A Java servlet produced XML from a proprietary middle tier and back end.
  • Notable features: Designed to be fully customizable by professional services staff in the field, targeted IE7+ (yeah, I know), UI embeddable within other products.
  • See it in action: Video at left walks through all major features, 90% of which I wrote.
  • More info: Decisiv Search
